## 单调性
设函数 $y=f(x)$的定义域为D,对于区间内任意两点$x_1$,$x_2$;<br>
当$x_1{<}x_2$的时候,恒有$f(x_1){<}f(x_2)$,则称函数$y=f(x)$在该区间内单调递增;<br>
当$x_1{<}x_2$的时候,恒有$f(x_1){>}f(x_2)$,则称函数$y=f(x)$在该区间内单调递减;
